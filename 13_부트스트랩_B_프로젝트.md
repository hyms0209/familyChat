- poc-bank 의 Router.ts, Provider, Tailwind 등 파일 복사 → 단계 C

## 실패 시 트러블슈팅

| 증상 | 원인 | 해결 |
|---|---|---|
| `npx create-expo-app` 에서 멈춤 | 네트워크 또는 npm cache | 채팅창에 "다른 SDK 버전으로 재시도" 또는 사용자가 수동으로 재시도 |
| 의존성 충돌 | 버전 불일치 | Claude 가 사용자에게 어느 버전 채택할지 묻도록 프롬프트에 이미 지시 |
| 한 응답 안에 다 못 끝냄 | Claude Code 한계 | 채팅창에 "의존성 설치 단계만 따로 다시 실행해줘" 부분 재시도 |
| poc-bank 의 Expo 버전이 너무 옛날 | poc-bank 가 오래된 SDK | "poc-bank SDK 가 너무 옛날이면 호환되는 가장 가까운 안정 버전 선택" 한 줄 추가해서 재시도 |
