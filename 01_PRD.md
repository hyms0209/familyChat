## 2. 개요

### 2.1 비전
가족이 자연스럽게 일상을 공유하고 함께 시간을 보낼 수 있도록 돕는 모바일 앱.

### 2.2 미션
**"흩어져 있는 가족 일상을 한 곳에"** — 위치, 일정, 대화를 가족만의 사적 공간에 모은다.

### 2.3 한 줄 정의
가족(2~4인 핵가족 중심) 전용 통합 커뮤니케이션·스케줄 앱. 가족 그룹 채팅, 실시간 위치 공유, 학원 일정을 포함한 공유 캘린더가 핵심.

### 2.4 작업명
**FamilyHub (가칭)** — 정식 명칭은 브랜딩 단계에서 확정.
대안 후보: 옹기종기 / 한데모여 / 우리네 / 모두마음 / 가족픽 / 우리집캘린더.

## 3. 배경 및 문제 정의

### 3.1 배경
한국 가족은 가족 일상을 관리하기 위해 다음과 같은 도구들을 분산 사용한다.

- **카카오톡 가족방** — 채팅, 사진 공유
- **TimeTree / Google Calendar** — 일정 공유
- **Find My / 새원가족** — 위치 공유
- **학원 단톡방, 학교 알림장 앱** — 자녀 일정/공지

이로 인해 정보가 흩어지고, 부모는 여러 채널을 매일 확인해야 한다.

### 3.2 사용자 Pain Points
1. **단편화**: 채팅, 일정, 위치가 각각 다른 앱에 있어 컨텍스트가 끊긴다.
2. **자녀 동선/일정 추적 부담**: 학원 셔틀 도착, 시간표 변경을 부모가 일일이 챙겨야 한다.
3. **사적 공간의 부재**: 카카오톡 가족방은 일반 채팅과 똑같이 보이고, 가족 컨텍스트가 사라진다.
4. **알림 피로**: 여러 앱에서 비슷한 알림이 중복 도착.

### 3.3 시장 기회
- 한국의 사교육 비중이 높고, 자녀 동선 관리에 대한 부모 수요가 명확.
- 한국어 UX와 학원/학교 컨텍스트에 특화된 통합 가족 앱은 부족.
- 가족 단위 구독 결제는 LTV가 높은 시장.

## 4. 목표 및 성공 지표

### 4.1 단계별 목표

| 단계 | 기간 | 목표 |
|---|---|---|
| 1차 (Personal Use) | 0~3개월 | 우리 가족(2~4인)이 일상적으로 사용 가능한 안정적 MVP |
| 2차 (Closed Beta) | 4~6개월 | 베타 사용자 50~100가족 확보, 핵심 가설 검증 |
| 3차 (Open Beta) | 7~9개월 | 500가족 + 주간 활성 가족(WAF) 70%+ |
| 4차 (Soft Launch) | 10~12개월 | 정식 출시, 유료 전환율 5%+ |

### 4.2 핵심 KPI

**활성도**
- DAU / WAU / MAU
- 가족당 평균 일별 메시지 수 (목표: 10+ msg/family/day)
- 위치 공유 활성 가족 비율 (목표: 70%+)
- 캘린더 일정 등록 수 (목표: 5+ events/family/week)

**리텐션**
- D1 / D7 / D30 리텐션
- 가족 단위 30일 리텐션 (목표: 60%+)

**비즈니스**
- 무료→유료 전환율 (목표: 5%+)
- 가족당 ARPU
- LTV / CAC

**품질**
- 푸시 알림 오픈율
- NPS (목표: 40+)
- 앱 크래시율 (< 0.5%)

## 6. 시장 분석 및 경쟁 분석

### 6.1 주요 경쟁 서비스

| 서비스 | 강점 | 약점 | 우리 차별점 |
|---|---|---|---|
| **Life360** | 위치 공유 강력, 운전 안전 분석, 글로벌 인지도 | 영어권 UX, 채팅 약함, 한국 학원/학교 컨텍스트 부재 | 한국형 학원/학교 컨텍스트 + 통합 채팅 |
| **새원가족 (삼성)** | 삼성 디바이스 통합, 무료 | iOS 한정적, UX 노후, 채팅·일정 제한 | 크로스플랫폼 + 채팅 중심 통합 UX |
| **카카오톡 가족방** | 채팅 강력, 보급률 거의 100% | 위치/일정 통합 부재, 가족 전용 구조 없음 | 가족 전용 구조 + 학원/위치 통합 |
| **아이쉐어 / 패밀리워치** | 자녀 위치/스마트폰 관리 | 부모-자녀 일방향, 채팅·캘린더 부재 | 양방향 가족 커뮤니티 + 자녀 보호 |
| **Famy / Famlinks** | 가족 사진 공유 | 채팅·위치·일정 부재 | 사진 외 핵심 기능 통합 |
| **TimeTree** | 가족 캘린더 강력 | 채팅·위치 부재 | 채팅·위치 통합으로 사용 빈도 락인 |

### 6.2 포지셔닝
- **세로축**: 위치 공유 강도 (실시간 ↔ 요청 시)
- **가로축**: 커뮤니케이션 강도 (단순 사진 공유 ↔ 채팅 중심)
- 우리는 **"강한 위치 공유 + 강한 채팅 + 한국형 학원 일정"**의 우상 영역에 위치.

### 6.3 차별화 전략
1. **한국형 학원 시간표 통합** — 학원명, 셔틀 시간, 픽업 위치를 캘린더에 자연스럽게 표기. 한국 1st 포지셔닝.
2. **자녀 친화 UX** — 위치 공유는 "프라이버시 존중하되 안전 우선" 톤. 일방적 감시감 제거.
3. **통합 알림 인박스** — 학원 도착, 일정 임박, 가족 메시지가 하나의 인박스로.
4. **가족 단위 데이터 격리** — 외부 SNS로 새지 않는 사적 공간.

## 7. 제품 범위

### 7.1 MVP (Phase 1, 0~3개월) — 우리 가족 사용 가능 수준

**필수 (Must Have)** — 기능 ID는 `02_기능명세서.md` §3 기준
- F1. 회원가입 / 로그인 (이메일·전화·소셜, 만 14세 미만 별도 흐름)
- F2. 가족 생성·초대·가입 (링크/QR)
- F3. 그룹 채팅 (텍스트·이모지·이미지)
- F4. 1:1 다이렉트 메시지
- F5. 실시간 위치 공유 (백그라운드 포함)
- F6. 공유 캘린더 (단발/반복 일정)
- F7. 학원 일정 / 시간표 통합 (반복 시간표 → 캘린더 자동 표기)
- F8. 푸시 알림 / 알림 인박스
- F9. 프로필 / 설정 (가족 관리, 데이터 권리 포함)

**제외 (Out of Scope for MVP)**
- 사진/동영상 앨범 → Phase 2
- 할 일/가사 분담 → Phase 2
- Geofence 기반 알림 → Phase 2
- 음성/영상 통화 → Phase 3
- 학원 결재/출석 체크 → Phase 3

### 7.2 Phase 2 (4~9개월) — Public Beta
- 사진/동영상 앨범 (월별·이벤트별 자동 정리, 원본 보관)
- 할 일/가사 분담 (체크리스트, 보상 포인트)

- 학원 픽업 알림 자동화 (위치 + 시간 결합)
- 홈 화면 위젯 (위치/오늘 일정)
- Freemium 결제 도입

### 7.3 Phase 3 (10개월+) — Soft Launch & 확장
- 음성 메시지 / 음성·영상 통화
- 자녀 디지털 웰빙 (스크린타임, 앱 사용 모니터링)
- 조부모 모드 (단순 UI)
- 가족 추억 타임라인 / 연도별 회상
- 학원 파트너십 (검색, 예약)
- 글로벌 (영어/일본어)

## 9. 비기능 요구사항 (NFR)

### 9.1 성능
- 채팅 메시지 송신→수신 도착 시간: **1초 이내** (정상 네트워크)
- 위치 갱신 주기: 30초~5분 (배터리·정확도 트레이드오프 자동 조정)
- 앱 콜드 스타트: 3초 이내
- 캘린더 월 뷰 로딩: 1초 이내

### 9.2 가용성
- MVP: 99.5%
- 정식 출시 후: 99.9%

### 9.3 보안
- 가족 간 데이터 완전 격리 (Multi-tenant, Firestore Security Rules 기반)
- 전송 구간 암호화 (TLS 1.2+)
- 저장 시 암호화 (Firebase 기본 + 민감 정보는 추가 암호화)
- 위치 정보는 가족 외부에 절대 노출되지 않음
- 인증: Firebase Auth (이메일, 전화번호, OAuth — Apple, Google, Kakao)
- 자녀 계정 보호: 부모 동의 + 자녀 단독 로그인 불가 옵션

### 9.4 개인정보보호 / 컴플라이언스
- **개인정보 보호법(PIPA)** 준수
- **만 14세 미만**: 법정대리인 동의 필요. 별도 가입 절차.
- **위치정보**: 「위치정보의 보호 및 이용 등에 관한 법률」 준수, 위치정보사업자 신고 검토.
- 데이터 최소 수집 원칙: 닉네임, 이메일, (선택)전화번호, (선택)프로필 이미지만.
- 이용자 권리: 열람/정정/삭제/이용정지 요청권 보장. 탈퇴 시 30일 내 완전 삭제.
- 개인정보처리방침/이용약관 별도 문서화 (출시 전 법무 검토 필수).

### 9.5 접근성
- 시스템 폰트 크기 대응 (Dynamic Type / Font Scale)
- 다크 모드 지원
- 색맹/약시 친화 컬러 팔레트 (WCAG AA 대비비)
- VoiceOver / TalkBack 지원

### 9.6 다국어
- v1: 한국어
- v2 (글로벌): 영어, 일본어 (시장 검증 후)

### 9.7 호환성
- iOS 15+ / Android 8+ (API 26+)

## 10. 수익화 모델

### 10.1 모델: Freemium

**무료 (Free)**
- 가족 1개, 최대 4명
- 기본 채팅, 실시간 위치 공유, 캘린더
- 학원 일정 최대 5개
- 사진 앨범 (Phase 2): 월 50장 + 자동 압축

**프리미엄 (Premium, 월 4,900원 / 연 49,000원 가정안)**
- 가족 최대 8명 (확장 가족 모드)
- 학원 일정 무제한 + 셔틀 자동 알림
- 사진 앨범 무제한 + 원본 보관
- 위치 히스토리 30일 → 1년
- 광고 제거 (무료 플랜에 한해 비침투형 배너)

### 10.2 단계별 도입
| Phase | 수익화 방식 |
|---|---|
| Phase 1 (MVP) | 무료, 우리 가족 사용 검증 우선 |
| Phase 2 (Beta) | Freemium 도입, 결제 흐름 검증 |
| Phase 3 (출시) | 본격 마케팅, 가족 단위 구독 |

### 10.3 부가 수익화 (Phase 3+ 검토)
- 학원 파트너십 (검색/예약 수수료) — 데이터 활용 윤리 가이드라인 필수

## 13. 디자인 / UX 원칙

### 13.1 톤 & 브랜드 무드
- **따뜻함, 부드러움, 안정감**
- 과하지 않은 컬러 팔레트 (예: 살구·연한 블루·아이보리)
- 둥근 모서리, 여유 있는 간격, 손글씨 느낌의 포인트

### 13.2 7가지 UX 원칙

1. **가족만의 사적 공간** — 외부 SNS와 명확히 구분되는 분위기.
2. **자녀의 자율성 존중** — 위치 공유에 자녀의 의사도 반영, 일방적 감시감 제거.
3. **단순함이 기본** — 처음 켰을 때 3초 안에 핵심 기능 발견.
4. **알림 절제** — 필요한 알림만, 중요도 기반 그룹화.
5. **컨텍스트 통합** — 채팅·위치·일정이 같은 카드/타임라인에 자연스럽게 섞임.
6. **접근성** — 폰트 크기, 다크 모드, 색약 친화.
7. **즐거움의 한 스푼** — 가족 이벤트(생일/기념일)에 자연스러운 축하 모먼트.

### 13.3 핵심 사용 시나리오

**시나리오 1 — 워킹맘의 점심**
> 지원이 점심에 앱을 켠다. 한 화면에 "오늘 16시 민서 영어학원 도착 예정", "오후 6시 가족 저녁", "남편 메시지 1건"이 보인다. 학원 도착 시 자동 알림이 도착한다.

**시나리오 2 — 자녀의 학원 종료**
> 민서가 학원 끝남. 한 번의 탭으로 "끝났어요. 데리러 와요" + 현재 위치 공유. 부모는 위치를 보고 차로 출발.

**시나리오 3 — 주말 약속 잡기**
> 토요일 점심을 가족이 같이 먹기로 한다. 캘린더에 일정 등록 → 가족 모두에게 알림 → 식당 위치 핀이 채팅에 자동 공유.
